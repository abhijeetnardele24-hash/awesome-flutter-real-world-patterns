# Theme Architecture

## Goal

Show a maintainable light/dark theme setup for a real Flutter app.

## Included in this example

- centralized theme definitions
- clean switching logic
- a dedicated `ThemeModeController`
- app-wide consistency

## Key files

- `lib/main.dart`
- `lib/app_theme.dart`
- `lib/theme_mode_controller.dart`

## Starter files

- `lib/theme/app_theme_mode.dart`
- `lib/theme/app_theme_controller.dart`
